# Tech Stack

The tech stack chosen for this project is the following

| Technology      | Role                             |
| --------------- | -------------------------------- |
| Next.js (React) | Front end framework              |
| Express.js      | Back end framework               |
| Node.js         | Runtime environment for back end |
| Postgresql      | Relational database              |

also known as **PERN**
