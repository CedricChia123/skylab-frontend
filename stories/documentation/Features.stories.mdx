# Features

This section documents features for the Skylab v2 platform as well as their status.

They are prioritized with numbers that are based on the [MoSCoW](https://www.agilebusiness.org/page/ProjectFramework_10_MoSCoWPrioritisation) priority system:

1. Must have
2. Should have
3. Could have
4. Won't have this time

---
